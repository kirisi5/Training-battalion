使用指南：
1、进入程序选择操作后会改变链表的值，这种改变不可逆，所以当输入错数据并且按了回车时，请重新打开应用。
2、程序还有些bug比如选项中的释放内存会导致一开始输入的链表消失，由于我没有把创建新链表作为一个选项所以，必须重新启动才能再次运行
3、创建的单链表都是带头结点的，并且不成环，所以会导致，判断是否成环永远都是不成环
4、反转链表操作打印不出来，请求老师和师兄帮看看我函数哪里出了问题（求赐教，不胜感激）
